MRCL

Info

Minecraft Replika of Code Lyoko (MRCL) is a project developed by a number of fans of the French show "Code Lyoko". I'm part of the admin team of the project, and as such I've worked on many Java plugins and Resource Pack assets for the server. Some examples are:

  • MC-Routines: A complex plugin whose objective is managing all the gameplay at the Kadic Academy. Its main focus is the NPCs - the plugin is able to spawn all the Kadic NPCs (students, teachers, workers...), and create a daily routine for them. Thanks to a external pathfinder, the NPCs make the school as lively as it can get!
  • ProjectCarthage: The main plugin of the server. I've done many contributions to it, such as working on class powers with advanced mechanics (terrain generation, object levitation, player triplication...), virtual interfaces, Lyoko Overvehicles...
  • PRC-SuperComputers: An extension to ProjectCarthage. This plugin is able to create a fully working Lyoko Terminal and Supercomputer. It allows you to interact with computer programs using left/right click, right from your player view.
  • MC-Monsters: A plugin able to generate XANA monsters and spawn them on the virtual worlds. They use custom NMS entities for their different behaviors.
  • AnimationAPI: An API used by MC-Monsters, it allows you to create an "Animation Skeleton" along with animations for it, in order to fully animate models in Java Edition!
  • Models: Apart from the code, I'm also the main modeler of the server. Some examples are the monsters, overvehicles, weapons, furniture...
  • And more! Chairs, elevators, skates, quests... The list just keeps growing :)
